Pengenalan HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. Dengan HTML, kita dapat menentukan struktur dari sebuah website, seperti heading, paragraf, gambar, tautan, tabel, dan elemen-elemen lainnya.

HTML terdiri dari elemen-elemen yang disebut dengan "tag". Tag ini digunakan untuk menginstruksikan browser tentang bagaimana konten harus ditampilkan.

Sejarah HTML

  • HTML 2.0
    (RFC1866) disetujui sebagai standard pada 22 September 1995
  • HTML 3.2
    14 Januari 1996
  • HTML 4.0
    18 Desember 1997
  • ISO/IEC 15445:2000 "ISO HTML"
    Berdasar pada HTML 4.01 Strict - 15 Mei 2000
  • XHTML 1.0 >> gabungan HTML & XML
    24 Januari 2000
  • XHTML 1.1
    16 Februari 2007

Ciri-ciri Dokumen

  1. Ekstensi file berupa .htm atau .html
  2. Non case sensitive
  3. Terdiri atas tag-tag pembuka dan penutup
  4. Tag-tag saling berpasang-pasangan

Struktur HTML

Struktur HTML terdiri dari document information, document header, dan document body. Document information terdiri dari tag html. Document header terdiri dari tag page titled, scripting, style, meta. Document body terdiri dari berbagai tag seperti teks & gambar, link, dll.
